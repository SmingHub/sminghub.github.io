SmingHub

SmingHub is the home of the Sming suite of C++ frameworks for the Espressif ESP8266 microcontroller.
An active community exists on gitter and each github project has a wiki and issue tracker.

Sming

Sming is a C++ framework for ESP8266.
Source code is available on github.

SmingRTOS

SmingRTOS is a C++ framework for ESP8266 using the Espressif RTOS platform.
